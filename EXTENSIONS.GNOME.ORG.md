## Description
```
Simple SSH extension to allow you to connect to your hosts at ~/.ssh/config and ~/ssh/know_hosts file from Gnome top bar

----- PREREQUISITE -----

1. Gnome 41
2. OpenSSH Client
3. Configured ~/.ssh/config file or existing ~/.ssh/known_hosts file
```
